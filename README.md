# Campus Navigation System

A comprehensive web-based navigation system designed to help students and visitors navigate campus buildings and locate facilities efficiently.

## Features

- **Interactive Campus Map**: Visual representation of campus layout
- **Building Search**: Quick search for specific buildings and facilities
- **Directions**: Turn-by-turn navigation between locations
- **Facility Locator**: Find classrooms, labs, offices, and amenities
- **Offline Support**: Access campus map without internet connection

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/campus-navigation-system.git
   ```

2. Navigate to the project directory:
   ```bash
   cd campus-navigation-system
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Mapping**: Leaflet.js / Google Maps API

## Usage

1. Open the application in your browser
2. Search for a building or facility using the search bar
3. Select your destination from the results
4. View the route and follow directions
